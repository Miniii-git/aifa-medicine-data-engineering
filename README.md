# AIFA Medicine Data Engineering
End-to-end data engineering project on Italian medicines dataset, AIFA data. Automated pharmaceutical data extraction, enrichment, database design, and analytics. Using Python, JavaScript, SQL, and Power BI.


## Overview

This project builds an enriched pharmaceutical dataset using official AIFA data.

The pipeline automatically downloads medicine PDFs, extracts clinical information, enriches the dataset, stores the data in a relational database, and visualizes the results through interactive Power BI dashboards.

## Technologies

- Python
- JavaScript (Node.js)
- SQL (MySQL)
- Power BI
- PDF Parsing

## Main Features

- Automated PDF download
- Clinical information extraction
- Dataset enrichment
- Relational database design
- Interactive dashboards
